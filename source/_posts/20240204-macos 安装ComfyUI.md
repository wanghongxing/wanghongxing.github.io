macos 安装ComfyUI



 https://github.com/comfyanonymous/ComfyUI



### Dependencies

Install the dependencies by opening your terminal inside the ComfyUI folder and:

```
conda activate comfyui
pip install -r requirements.txt
```



#### Run

```
python main.py --force-fp16
```
